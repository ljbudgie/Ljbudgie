### 🔭 What I'm Building

| Project | Description |
|---------|-------------|
| [**The Burgess Principle**](https://github.com/ljbudgie/burgess-principle) | A minimalist, human-first protocol for demanding individual scrutiny over automated decisions. Calm templates, open-source Python toolkit, and an optional Sovereign Personal Vault. |
